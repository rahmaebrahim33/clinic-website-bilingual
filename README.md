# 🏥 Advanced Bilingual Clinic Website

A modern, fully responsive medical clinic website with bilingual support (Arabic & English), smooth animations, and a professional healthcare-focused UI/UX.

---

## 🌐 Live Demo

🔗 https://rahmaebrahim33.github.io/clinic-website-bilingual/

---

## 📌 Overview

This project is a professional clinic website designed to simulate a real-world healthcare platform.
It focuses on clarity, trust, and ease of appointment booking while supporting both Arabic and English users with dynamic language switching and RTL/LTR layout support.

---

## ✨ Key Features

### 🌍 Bilingual System

* Full support for English and Arabic
* Dynamic language switching without page reload
* Automatic layout direction change:

  * English → LTR
  * Arabic → RTL

---

### 🎬 Smooth Animations

* Fade-out and fade-in transitions when switching language
* Subtle scroll animations for sections
* Hover effects for buttons and cards
* Clean and minimal motion design

---

### 🩺 Appointment System

* Step-based booking form:

  1. Select Doctor
  2. Choose Date
  3. Enter Patient Details
* Confirmation message after submission
* Smooth transition between steps

---

### 👨‍⚕️ Doctors Section

* Clean doctor profiles layout
* Specialization and experience displayed
* Easy “Book Appointment” action

---

### 📋 Services Section

* Simple and structured list format:

  * General Consultation
  * Dental Care
  * Pediatrics
  * Cardiology
  * Lab Tests

---

### 📞 Contact Section

* Clinic contact details
* Working hours
* Location information
* Map placeholder

---

## 🎨 UI/UX Design
